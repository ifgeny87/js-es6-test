Хитрости использования JS

  • Конвертация в Boolean с "!!" 
    console.log(!!1);	// true
console.log(!!'a');	// true
console.log(!!'');	// false
console.log(!!false);	// false
console.log(!!NaN);	// false
console.log(!!null);	// false
console.log(!!undefined);	// false
console.log(!!Infinity);	// true
  • Конвертация в Number с "+" 
    console.log(+1);	// 1
console.log(+'1');	// 1
console.log(+'a2');	// NaN
console.log(+'1e3');	// 1000
console.log(+'.5');	// 0.5
console.log(+'0x33');	// 51
console.log(+new Date());	// 1521061818617
  • Упрощеная запись условия с "&&"
  • Значения по умоланию с "||"
  • Слияние массивов с Array.concat
  • Конвертация в массив с Array.from
  • Перемешивание массива 
    [1,2,3,4].sort(() => Math.random() - .5);	// 2,1,4,3
